## 文档规范

### 文档命名
- 使用中文描述性名称
- 括号内注明简称：`(PRD)`、`(TDD)`
- 使用 `.md` 扩展名

### 文档格式
- 使用 Markdown 格式
- 使用标准标题层级（H1-H4）
- 代码块指定语言类型

### 变更记录
- 每次更新在文档顶部或专门章节记录
- 包含日期、版本号、变更内容
- 保持变更记录的可追溯性
