# Global Macro Finance - 后端技术规范文档

## 1. 概述

### 1.1 背景
构建一个独立的 Python FastAPI 后端服务，用于获取和提供全球宏观经济债券利率数据（美债、欧债、日债），供 n8n 工作流和前端展示使用。

### 1.2 目标
- 提供美债、欧债、日债利率数据的获取和存储
- 支持 n8n 主动调用触发数据更新
- 提供前端数据查询接口
- 作为 Personal-Web 项目的一个独立微服务部署

### 1.3 非目标
- 本规范仅涵盖后端实现，前端展示功能后续单独讨论
- 暂不包含汇率数据（后续扩展）

## 2. 功能需求

### 2.1 核心功能
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 数据获取 | 从 FRED API 获取美债、欧债、日债利率数据 | P0 |

| 数据查询 | 提供接口查询历史数据，支持时间范围过滤 | P0 |

### 2.2 API 接口规范

#### POST /api/update
**描述**: n8n 调用此接口触发数据更新，获取最新数据并追加到 CSV

**请求**: 无需请求体

**响应**:
```json
{
  "success": true,
  "message": "数据更新成功",
  "data": {
    "us_treasuries": {
      "3m": {"date": "2025-01-15", "value": 4.25},
      "2y": {"date": "2025-01-15", "value": 4.15},
      "10y": {"date": "2025-01-15", "value": 4.05}
    },
    "eu_10y": {"date": "2025-01-15", "value": 2.65},
    "jp_10y": {"date": "2025-01-15", "value": 0.75}
  },
  "updated_at": "2025-01-15T10:30:00Z"
}
```

**错误响应**:
```json
{
  "success": false,
  "message": "数据更新失败：API 调用超时",
  "error_code": "API_TIMEOUT"

```

## 7. 边缘情况与错误处理

### 7.1 异常场景处理

| 异常场景 | 处理策略 |
|----------|----------|
| FRED API 网络超时 | 自动重试，最多 3 次，每次延迟递增 |
| FRED API 返回错误 | 记录错误日志，返回错误响应 |
| CSV 文件不存在 | 首次访问时自动创建文件头 |
| CSV 文件损坏 | 记录错误，返回空数据 |
| 并发更新请求 | 使用锁机制，串行执行，后续请求等待或返回状态 |

### 7.2 数据验证
- 日期格式验证 (YYYY-MM-DD)
- 数值范围验证（利率通常在 0-20 之间）
- 缺失值处理（前向填充）

### 7.3 重试机制
```python
import asyncio
from functools import wraps

def async_retry(max_retries=3, delay=1.0):
    def decorator(func):
        @wraps(func)
        async def wrapper(*args, **kwargs):
            for attempt in range(max_retries):
                try:
                    return await func(*args, **kwargs)
                except Exception as e:
                    if attempt == max_retries - 1:
                        raise
                    await asyncio.sleep(delay * (2 ** attempt))
        return wrapper
    return decorator
```

### 7.4 并发控制
```python
from asyncio import Lock

class UpdateLock:
    _lock = Lock()
    _is_updating = False

    @classmethod
    async def acquire(cls):
        await cls._lock.acquire()
        cls._is_updating = True

    @classmethod
    def release(cls):
        cls._is_updating = False
        cls._lock.release()

    @classmethod
    def is_updating(cls) -> bool:
        return cls._is_updating
```

### 8.3 Gateway 路由配置
在 Personal-Web Gateway 添加路由：
```python
# gateway/app/routers/macro.py
from fastapi import APIRouter
from httpx import AsyncClient

router = APIRouter(prefix="/api/macro", tags=["macro"])

MACRO_SERVICE_URL = "http://global-macro-fin:8094"

@router.post("/update")
async def update_data():
    async with AsyncClient() as client:
        response = await client.post(f"{MACRO_SERVICE_URL}/api/update")
        return response.json()

@router.get("/data")
async def get_data(start_date: str = None, end_date: str = None):
    async with AsyncClient() as client:
        params = {"start_date": start_date, "end_date": end_date}
        response = await client.get(f"{MACRO_SERVICE_URL}/api/data", params=params)
        return response.json()
```

### 9.1 技术风险
| 风险 | 等级 | 缓解措施 |
|------|------|----------|
| FRED API 配额限制 | 中 | 记录 API 调用次数，设置阈值告警 |

### 9.2 外部依赖
- **FRED API**: 需要有效的 API 密钥
- **网络连接**: 需要稳定的互联网连接访问 FRED API
- **磁盘空间**: CSV 文件会持续增长，需要监控磁盘使用
