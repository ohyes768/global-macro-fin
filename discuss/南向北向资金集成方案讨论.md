# 南向/北向资金集成方案讨论文档

## 1. 需求概述

### 1.1 用户需求
在宏观经济数据服务中新增南向资金（港股通）和北向资金（沪深港通）数据模块，为前端提供资金流向数据接口，支持 n8n 自动化工作流定时更新数据。

### 1.2 关键要求
- 数据来源：AKShare（免费开源）
- 更新频率：每日更新
- 历史数据：从 2014-11-17 开始（沪港通开通日）
- 存储方式：CSV 文件存储
- 数据内容：北向资金净流入、南向资金净流入及买卖成交额

## 4. 关键技术决策

### 4.1 AKShare 接口选择

**决策**: 使用东方财富数据源接口

**理由**:
- 数据更新及时
- 历史数据完整
- 接口稳定

**接口选择**:
```python
# AKShare 接口示例
ak.stock_hsgt_north_net_flow_in_em()  # 北向资金净流入
ak.stock_hsgt_south_net_flow_in_em()  # 南向资金净流入
```

---

### 4.2 并发控制

**决策**: 复用现有的全局锁机制

**理由**:
- 保持并发控制逻辑统一
- 避免并发更新导致的数据不一致
- 简化代码逻辑

**实现**:
```python
# 复用现有的锁机制
if _is_updating:
    return UpdateResponse(
        success=False,
        message="数据更新正在进行中，请稍后再试",
        error_code="UPDATE_IN_PROGRESS"
    )
```

---

### 4.3 数据对齐

**决策**: 与现有美债数据日期对齐，非交易日不填充数据

**理由**:
- 资金流向数据仅在交易日有数据
- 与美债数据对齐便于前端展示
- 保持数据的真实性

**实现**:
```python
# 前向填充非交易日数据（可选）
data = data.ffill()
```

---

### 4.4 历史起始日期

**决策**: 从 2014-11-17 开始（沪港通开通日）

**理由**:
- 这是北向/南向资金数据的起始日期
- 可获取约 11 年完整历史数据
- 数据量适中（约 2700+ 个交易日）

---

### 4.5 更新策略

**决策**: 增量更新最近 7 天

**理由**:
- 与美债/汇率的更新策略保持一致
- 覆盖周末和节假日，确保数据完整性
- 响应时间可控
