## 3. API 数据模型

### 3.1 TreasuryData (国债数据点)

```python
class TreasuryData(BaseModel):
    date: date           # 数据日期
    value: Optional[float] = None  # 收益率值（可能为空）
```

**说明**: 表示单个国债在某日期的收益率数据点

### 3.2 USTreasuries (美国国债数据)

```python
class USTreasuries(BaseModel):
    m3: TreasuryData    # 3月期国债
    y2: TreasuryData    # 2年期国债
    y10: TreasuryData   # 10年期国债
```

**说明**: 包含三种期限的美国国债数据

### 3.3 USTreasuriesUpdateData (美债更新响应)

```python
class USTreasuriesUpdateData(BaseModel):
    us_treasuries: USTreasuries
```

**说明**: 美债更新接口的响应数据，只包含美债

### 3.4 MacroData (宏观经济数据)

```python
class MacroData(BaseModel):
    us_treasuries: USTreasuries
    eu_10y: TreasuryData   # 德国10年期国债
    jp_10y: TreasuryData   # 日本10年期国债
```

**说明**: 完整的宏观经济数据响应

n### 3.5 VIXData (VIX恐慌指数数据)

```python
class VIXData(BaseModel):
    date: date           # 数据日期
    value: Optional[float] = None  # VIX指数值（可能为空）
```

**说明**: 表示VIX恐慌指数在某日期的数值

### 3.6 VIXUpdateData (VIX更新响应数据)

```python
class VIXUpdateData(BaseModel):
    vix: VIXData
```

**说明**: VIX更新接口的响应数据

## 5. 数据模型变更记录

### 2026-03-03 - v1.0.0

#### 新增数据模型

**USTreasuriesUpdateData**
- **变更类型**: 新增
- **用途**: 美债更新接口专用响应模型
- **字段**:
  - `us_treasuries: USTreasuries` - 美债数据
- **原因**: 美债更新接口不需要返回 OECD 债券数据

#### 修改数据模型

**UpdateResponse.data**
- **变更类型**: 修改
- **原类型**: `Optional[MacroData]`
- **新类型**: `Optional[USTreasuriesUpdateData | MacroData]`
- **原因**: 支持美债更新接口返回不同的数据结构

#### 新增数据文件

**数据文件初始化**
- **us_treasuries.csv**: 美债历史数据（2000-01-01 至今）
- **eu_bonds.csv**: 德国债券数据
- **jp_bonds.csv**: 日本债券数据
- **数据量**: 美债约 6800+ 条，OECD 约 300+ 条

## 6. 数据质量规则

### 6.1 数据完整性
- 日期连续性：美债每日数据，允许节假日缺失
- 缺失值处理：使用 null 填充，前端前向填充
- 去重规则：同一日期保留最新获取的数据

### 6.2 数据准确性
- 数据来源：仅来自 FRED 官方 API
- 异常值：保留原始值，不人工修正
- 精度：保留 2 位小数

### 6.3 数据时效性
- 更新频率：美债每日，OECD 月度
- 延迟：数据延迟 1 个工作日
- 缓存：无缓存，实时查询

## 8. 数据字典使用说明

### 8.1 前端开发
- 日期字段使用字符串格式 "YYYY-MM-DD"
- 收益率值为浮点数，单位：百分比
- 缺失值为 null，需要前向填充处理

### 8.2 后端开发
- 使用 Pandas 读取 CSV 文件
- 日期列作为索引
- 使用 ffill() 前向填充缺失值

### 8.3 数据分析
- 美债数据质量高，每日更新
- OECD 数据为月度，早期可能缺失
- 注意时区和交易日历差异
